ROLI Seaboard Web Emulator

Welcome to the ROLI Seaboard Web Emulator, a browser-based application that recreates the innovative touch-sensitive keyboard experience of the ROLI Seaboard hardware. This project allows musicians and enthusiasts to explore the expressive capabilities of the Seaboard's unique interface directly in your web browser.

Original Version Enhanced Version Documentation

About the Project

The ROLI Seaboard is known for its revolutionary "5D Touch" system that allows for unprecedented musical expression through touch. This web emulator recreates that experience using modern web technologies, enabling you to play with strike, press, glide, slide, and lift interactions.

Key Features

  • 5D Touch emulation (strike, press, glide, slide, lift)
  • Polyphonic synthesizer with multiple oscillator types
  • Audio effects including reverb and delay
  • Recording and export capabilities
  • Multi-touch support on compatible devices
  • Responsive design for desktop and mobile
